# Image Instructions

## Horse Logo

Please save your horse logo image as:
`public/horse-logo.png`

The image should be:
- A shield/badge design with a horse
- Contains the text "馬來迎富"
- PNG format with transparent background (recommended)
- Recommended size: 512x512px or larger for best quality

The app will automatically use this image in the header on both sides of the title.

If you uploaded the image in the chat, you can:
1. Download it from the chat
2. Save it to `public/horse-logo.png`
3. Restart the dev server

Current path expected: `/horse-logo.png` (which maps to `public/horse-logo.png`)
